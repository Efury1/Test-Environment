# GitHub Guide

GitHub is the leading platform for hosting, reviewing, and collaborating on code. This guide will cover its benefits, key workflows, and pros and cons, with visuals to illustrate each feature.

---

## Benefits of GitHub

### 1. Repository Management
GitHub makes it easy to create, organize, and manage code repositories.

![Add GitHub Repo link](images/Github/Github_Add_Github_Repo_link.png)

### 2. Branching and Workflow Rules
GitHub supports conventional branch naming, protections, and workflow rules to enforce code quality.

![Branch conventional rules](images/Github/Github Branch conventional rules.png)  
![Branch protections](images/Github/github_branch_protections.png)

### 3. Pull Requests and Code Review
GitHub Pull Requests (PRs) allow team collaboration, review, and automated checks before merging code.

![Co-pilot PRs](images/Github/Github_co-pilot_PRs.png)  
![Copilot code review workflow](images/Github/Copilotcode review workflow.png)  

### 4. GitHub Actions
Automate workflows like CI/CD, tests, and deployments directly from GitHub.

![Different action statuses](images/Github/Github Different action statuses.png)  
![Failing github action](images/Github/Failing github action.png)  
![Run workflow](images/Github/Gitub_run_workflow.png)  
![Deploy run wiki](images/Github/Github deploy run wiki.png)

### 5. Integration with Azure
GitHub integrates seamlessly with Azure DevOps for boards and work item tracking.

![Connect GitHub with Azure](images/Github/Connect_Github_with_Azure.png)  
![Connect Azure boards](images/Github/Github_Connect_Azure_boards.png)  
![Azure search work items](images/Github/Github_with_Azure_search_work_items.png)

### 6. Collaboration Features
GitHub provides wikis, discussions, and free plans to encourage collaboration and knowledge sharing.

![GitHub Wiki](images/Github/Github_Wiki.png)  
![GitHub Discussions](images/Github/Github_Discussions.png)  
![Free Plan](images/Github/Github Free Plan.png)

### 7. Merge Controls
GitHub allows controlling merge behavior, including checks, approvals, and waiting for requirements.

![Merge without waiting](images/Github/Github_merge_without_waiting_for_requirments.png)  
![Some checks haven't completed](images/Github/Github_some_checks_havent_completed.png)

### 8. GitHub Copilot
GitHub Copilot helps developers write code faster by suggesting AI-powered completions.

![Githut copilot](images/Github/Githut_copilot.png)

---

GitHub is a robust platform for both individuals and teams. By leveraging branching rules, PR reviews, actions, and integrations, teams can maintain high code quality and accelerate development. It has an extensive marketplace for CI/CD pipelines via Github actions.
